# LotteryPicker

A Ruby gem to generate lottery drawings.

## Installation

Add this line to your application's Gemfile:

    gem 'lottery_picker'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lottery_picker

## Usage

ruby
LotteryPicker::Drawing.new.draw # => [23 ,2 ,12, 4, 3]
